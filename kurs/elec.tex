\section{Реализация регулятора}
Для реализации регулятора, представим его передаточную функцию
в виде произведения типовых звеньев, для которых уже разработаны элек-
трические схемы [1] и соединим их последовательно. Так как коэффициент
q не сильно меньше 1, пренебрежём им и получим передаточную функцию
регулятора следующего вида:
\begin{align}
W_p(s) = \frac{K_p(T_2s + 1)(T_1s + 1)^2}{s(T_3s + 1)^2}.
\end{align}

Разбив на множители, получим

\begin{align}
W_p(s) = \frac{T_2s + 1}{T_0s}\cdot \sqrt{K_p}\frac{T_1s + 1}{T_3s + 1}\cdot \sqrt{K_p}\frac{T_1s + 1}{T_3s + 1}.
\end{align}

Тогда электрическая схема регулятора принет вид, представленный
на рисунке \ref{elchem}.
\begin{figure}[ht!]
	\centering
	\includegraphics[width = 0.9\textwidth]{img/elchem}
	\caption{Электрическая схема реализации регулятора}
	\label{elchem}
\end{figure}

На представленной схеме выполняются равенства
\begin{align}
\begin{cases}
R_3 = R_5\\
R_4 = R_6\\
C_2 = C_3
\end{cases}
\end{align}

Рассчитаем значения элементов схемы.
\begin{align}
\begin{cases}
\begin{cases}
T_0 = R_1C_1 = 1\\
T_2 = R_2C_1 = 0.059
\end{cases}\\
\begin{cases}
T_1 = {R_3}{C_1} = 0.22\\
T_3 = \frac{R_3*R_4}{R_3+R_4}*c_2 = 0.0067\\
\sqrt{K_p}= \frac{R_4}{R_3+R_4} = 0.066\\
\end{cases}\\
\end{cases}
\end{align}

\par
Как можно заметить, в первой системе 2 уравнения и 3 неизвестных, поэтому решим эту систему, приняв $R_2 = 1 Ом$. Тогда получим

\begin{gather}
\begin{cases}
\begin{cases}
R_2 = 1 \text{Ом}\\
C_1 = \frac{0.059}{1} = 0.059 \text{ Ф}\\
R_1 = \frac{1}{0.059} = 16.94 \text{ Ом}
\end{cases}\\
\begin{cases}
C_2 = \frac{0.22}{R_3} \text{ Ф}\\
R_4 = 0.07R_3 \text{ Ом}\\
\frac{R_6*0.07R_6}{R_6+0.07R_6} * C_2 = 0.0067 \\
\end{cases}
\end{cases}
\end{gather}

Решив вторую систему получим следующие значения для всех эле-
ментов схемы:
\begin{gather}
\begin{cases}
R_1 = 16.94 \text{ Ом}\\
R_2 = 1 \text{ Ом}\\
R_3 = R_5 = 2 \text{ Ом}
R_4 = R_6 = 0.14\text{ Ом}\\
C_1 = 0.059 \text{ Ф}\\
C_2 = C_3 = \text{ Ф}\\
\end{cases}
\end{gather}

Схема была составлена в среде разработки P-CAD 2006.
